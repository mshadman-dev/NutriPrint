🌿 About NutriPrint

NutriPrint

AI-Powered Nutrition Awareness Platform for Karnataka Schools

NutriPrint helps schools identify nutritional gaps, generate balanced meal plans, promote healthy eating habits, and improve nutrition awareness among children.

Mission
Empower schools through data-driven nutrition
NutriPrint's mission is to make nutrition science accessible to every Karnataka school teacher — turning raw student data into clear, actionable health guidance without requiring clinical expertise.
⚠️
Problem Statement
Schools lack simple, integrated nutrition tools
Childhood malnutrition, poor dietary awareness, absence of structured guidance, and difficulty planning balanced affordable meals leave teachers without an effective toolkit for student health.
✅
Solution Overview
One platform covering the full nutrition workflow
From BMI assessment to AI meal planning, printable posters, QR recipes, and bilingual parent guidance — NutriPrint closes every gap in a single cohesive health-tech workspace.

Technology Stack

Built on a modern production stack

Every layer chosen for reliability, performance, and developer experience.

🐍
Language
Python
Core application language — clean, readable, and supported by the richest data science and AI ecosystem.
⚡
Backend Framework
FastAPI
Async REST API with automatic OpenAPI docs, Pydantic validation, and structured modular routers.
🤖
AI Engine
Groq API
Ultra-fast LLM inference for meal plan generation, bilingual nutrition advice, and AI recommendations.
🗄️
Database
SQLite · Supabase
Local SQLite for development; Supabase Postgres in production with auth, real-time, and row-level security.
🌐
Frontend
HTML · CSS · JavaScript
Tailwind CSS, Jinja2 templates, vanilla JS — zero framework overhead, maximum control.
